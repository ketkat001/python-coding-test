## 그리디 알고리즘

### 정의 :

​	그리디 알고리즘 (욕심쟁이 알고리즘, Greedy Algorithm)은 단순하지만 강력한 해결 방법이다.  어떠한 문제가 있을 때 단순 무식하게 탐욕적으로 문제를 푸는 알고리즘. 여기서의 탐욕의 뜻은 "현재 상황에서 지금 당장 좋은 것만 고르는 방법"을 의미.

​	그리디 알고리즘으로 문제를 해결할 경우 그 해법이 정당한지 검토해야 한다.
